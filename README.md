# GitHub Pages upload

1. Create a GitHub repository.
2. Upload all files in this folder.
3. In GitHub, open Settings → Pages.
4. Under “Build and deployment”, choose:
   - Source: Deploy from a branch
   - Branch: main
   - Folder: / (root)
5. Save.

Your site should then publish automatically.

If you want the homepage to appear at the root URL, keep the file named `index.html`.
